## 🚀 About This Project

Welcome to the source code of my personal portfolio website! This project serves as a central hub to showcase my journey as a computer science student at NIT Durgapur, my technical projects, and my active involvement in open-source software. 

The website is designed with a modern, glassmorphic UI, smooth micro-animations, and a responsive layout to ensure a premium user experience across all devices.

### ✨ Key Features
- **Modern & Responsive UI**: Clean aesthetics utilizing glassmorphism, dynamic background blobs, and smooth fade-in animations.
- **Interactive Sections**: Neatly categorized sections for Education, Open Source Contributions, Projects, Skills, and Achievements.
- **Optimized for Performance**: Built with Vite for lightning-fast HMR and optimized production builds.

## 🛠️ Built With

* **Core**: Vanilla HTML5, CSS3, JavaScript (ES6+)
* **Tooling**: [Vite](https://vitejs.dev/)
* **Environment**: [Nix](https://nixos.org/) (Flakes configured for reproducible dev environments)
* **Hosting**: [Vercel](https://vercel.com/) (Ready for seamless deployment)

## 💻 Getting Started Locally

To get a local copy up and running, follow these simple steps:

### Prerequisites

You need `Node.js` and `npm` installed. If you use Nix, you can simply load the development shell via the provided `flake.nix`.

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/Tejas-Raj01/portfolio.git
   ```
2. **Navigate to the directory**
   ```sh
   cd portfolio
   ```
3. **Install NPM packages**
   ```sh
   npm install
   ```
4. **Start the development server**
   ```sh
   npm run dev
   ```
5. **Open your browser**
   Navigate to `http://localhost:5173` (or the port Vite provides) to see the app running locally!
